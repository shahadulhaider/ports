## Implementation Summary
- Cache stored at os.UserCacheDir()/ports/last.json
- Identity key: (Port, PID, Process) tuple
- Exit codes: 0=no changes/first run, 1=changes detected, 2=error
- Port filter support via portFilter parameter
- Sorted output for determinism
